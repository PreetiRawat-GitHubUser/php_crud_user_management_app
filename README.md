Project Title :
PHP CRUD User Management App

Description :

This is a simple User Management application built using core PHP and AJAX.
The application performs basic CRUD operations (Create, Read, Update, Delete)
without using any PHP framework.

The project demonstrates usage of:
- Procedural PHP concepts
- AJAX for asynchronous operations
- MySQL database integration
- One-page application behavior (no page reload)

FEATURES :

- Add new user using AJAX
- Display users in a table without page reload
- Edit user details using AJAX
- Delete user records using AJAX
- One-page split layout (Form + User List)

DB SETUP

1. Create a MySQL database
2. Import the SQL file located in the `db/database.sql` folder
3. Update database credentials in the configuration file

How To Run Project

1. Install XAMPP or any PHP server
2. Place the project inside the `htdocs` directory
3. Start Apache and MySQL
4. Open browser and navigate to:
   http://localhost/user-list-app

AJAX WORKFLOW

- Form submission is handled using JavaScript and AJAX
- Data is sent to PHP scripts without reloading the page
- PHP scripts interact with the database
- Updated user list is returned and rendered dynamically
